# GNN Terms Popup

A lightweight, one-time mandatory Terms & Conditions popup plugin for WordPress.

## Features
- **Mandatory Acceptance:** Users must accept terms before proceeding.
- **Cookie-based:** Persists acceptance state via browser cookies.
- **Dynamic Content:** Pull legal text from a WordPress Page or enter it manually.
- **Display Scope:** Configure exactly which paths should trigger the popup.
- **GNN Premium UI:** Professional yellow/black theme with high accessibility.

## Documentation
A detailed [Changelog](CHANGELOG.md) is maintained in the root directory.

## Installation
1. Upload the plugin folder to `/wp-content/plugins/`.
2. Activate via the WordPress Admin panel.
3. Configure settings under **Settings -> GNN Terms Popup**.
